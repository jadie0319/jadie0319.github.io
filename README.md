# Stock Service

## 목적

- Kotlin 공부
- AI 도구 활용 기능 개발

## 사용기술
- Kotlin
- Spring boot 3.5.7
- Spring data jdbc
- Spring cloud openfeign, resilience4j
- H2, MariaDB
- ChatGpt or Claude or Gemini

## 핵심기능
- 데이터 수집(collect)
- 데이터 분석(analyze)
- 데이터 노출(report)
